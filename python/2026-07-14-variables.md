# 변수와 자료형

📅 2026-07-14

🏷️ Python, 변수, 자료형

## 오늘 배운 것
✅ 변수 선언 및 자료형 (int, float, str, bool)

✅ 숫자 연산 (+, -, *, /, //, %, **)

✅ 문자열 연산 (+, *, len())

✅ f-string 출력

✅ 에러 읽는 법 (NameError)

## 헷갈렸던 점
⚠️ / 랑 // 차이

→ /는 소수점까지, //는 몫만 나옴

⚠️ 네 가지 변수 한번에 출력

→ f-string 사용 (f"이름: {name}")

## 내일 할 것
