# 반복문

📅 2026-06-22
🏷️ Python, 반복문

## 오늘 배운 것
✅ for문과 range()

for은 정해진 횟수만큼 반복
```
for i in range(5):
    print(i)
```

```range()
range(5)      → 0, 1, 2, 3, 4      (0부터 4까지)
range(1, 6)   → 1, 2, 3, 4, 5      (1부터 5까지)
range(0, 10, 2) → 0, 2, 4, 6, 8   (2씩 증가)
⚠️ 시작은 포함, 끝은 미포함
```

✅ 리스트 반복

range 없이도 리스트만 있으면 반복 가능
```
sensors = [10.5, 20.3, 15.7, 8.2]

for value in sensors:
    print(value)
```

✅ while문

"조건이 True인 동안" 계속 반복
```
speed = 0

while speed < 5:
    print(f"현재 속도: {speed}")
    speed += 1
```

✅ break와 continue

break    → 반복문 완전 탈출
continue → 이번 것만 건너뛰고 다음으로
```
for i in range(10):
    if i == 5:
        break       # i가 5가 되면 반복 종료
    print(i)

# 출력: 0, 1, 2, 3, 4
```
```
for i in range(10):
    if i % 2 == 0:
        continue    # 짝수면 건너뜀
    print(i)

# 출력: 1, 3, 5, 7, 9
```

✅️ 들여쓰기

Python은 들여쓰기로 코드 소속 구분
```
for i in range(1, 100):
    if i % 3 > 0:
        continue
    if i > 50:
        break
    print(i)   # ⭕️ for문 안 → 반복됨

print(i)    # ❌️ for문 밖 → 반복 끝나고 한 번만 실행
```

## 헷갈렸던 점
⚠️ 3의 배수값을 구하고 싶으면, i % 3 == 0

⚠️ 들여쓰기로 소속 구분
